# User Stories

```text
- As a customer, so I can receive my tickets, I want to provide my contact information.
- As a customer, so I can decide which movie I want to watch, I want to see a list of movies.
- As an admin, so I can manage the movies shown at the cinema, I want to update the list of movies.
- As a customer, so I can choose a specific movie, I want to see the available showtimes.
- As a customer, so I can book tickets for a movie, I want to select the desired showtime.
- As a customer, so I can keep track of my bookings, I want to view a history of my ticket purchases.
- A single screen might show multiple movies a day, and even the same movie at multiple times.
- The cinema will expand its number of screens in the future, so the potential for growth needs to be accounted for.
```
